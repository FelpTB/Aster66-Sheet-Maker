# Blefe

Alcançe: Si Mesmo
Modo de Uso: Passivo
Origem: Cartomante
Tipo: Habilidade

Ao sacar uma carta você não a revela e pode apenas falar qual o resultado, caso o mestre queira ele pode duvidar, se ele acerta o blefe o resultado do saque é considerado um fracasso automático no teste.

Caso esteja usando “Ataque de Cartas” o dano se torna 0 quando o blefe for pego.

Caso esteja usando “Mão Quente” a carta explode te causando 1d6 de dano.

Caso esteja usando “Jackpot” a sequência é quebrada.
