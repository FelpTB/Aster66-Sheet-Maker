# Ordem Rápida

Alcançe: Em Cena
Modo de Uso: Reação
Origem: Cavaleiro
Tempo de Uso: Imediato
Tipo: Habilidade

Descreve uma ordem rápida para um aliado, essa ordem deve poder ser cumprida no tempo de uma reação, se ele quiser seguir a ordem ele não gasta reações para isso.

- Você basicamente da uma reação extra para um aliado, decidindo o que ele deve fazer com ela.
- Pode ser usado inclusive para fazer aliados bloquearem ou esquivarem.
