# Clone das Sombras

Alcançe: Corpo a Corpo
Condição de aprendizado: Ligação += 5
Custo: 5 de energia
Modo de Uso: Ação
Origem: Magia Proibida, Magia das Sombras, Pacto das Sombras
Tempo de Uso: Imediato
Tipo: Magia

O usuário cria um clone feito de sombras que toma 5 de mana do usuário temporariamente, caso o clone seja acertado por um ataque ele se desfaz instantaneamente devolvendo 2 de mana para o criador.

Clone das Sombras: Possui 1 ação por turno e causa 1d6 de dano de ataque (De corte, De impacto ou de perfuração O usuário escolhe), o clone obedece a vontade do criador e pode realizar ações simples que não exigem testes. (Clones contam como unidade invocada/ Corpo Vazio.). O clone é imune a condições.

Efeito Colateral: O usuário recebe o dano do ataque que desfizer os clones.
