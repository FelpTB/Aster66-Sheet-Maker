# Domínio do Espadachim

Custo: 3 de energia
Modo de Uso: Ação
Origem: Samurai
Tipo: Habilidade

Enquanto estiver usando uma Espada:

Entra em postura de saque não podendo se mover enquanto mantiver a habilidade ativa. Nessa postura a área de até 1 métro adjascente a você (3x3 com você no meio) é considerada seu domínio. Sempre que algo invadir seu domínio, nesse momento, você pode gastar uma reação para golpear instantaneamente o objeto ou criatura invasora. Se for um projétil mesmo que mágico ele é completamente defletido, se for uma criatura ela é golpeada por um ataque de saque.

Essa habilidade encerra caso você se mova do centro do domínio.

Explicação:

Os projéteis são defletidos intantanemente sem a necessidade de um teste de aparar.

Os golpes dessa habilidade ativam a habilidade Iaidô sem necessidade de voltar a espada para a bainha.

Só é possível golpear uma criatura com uma reação uma vez usando essa habilidade e isso deve ocorrer no exato momento em que ela invade a área do domínio, golpes subsequentes devem ser realizados normalmente.
