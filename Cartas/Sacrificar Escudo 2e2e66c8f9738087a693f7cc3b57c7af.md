# Sacrificar Escudo

Alcançe: Corpo a Corpo
Custo: 3 de energia
Modo de Uso: Reação
Origem: Cavaleiro
Tempo de Uso: Imediato
Tipo: Habilidade

Destrói o escudo que estiver equipado mas anula completamente o dano de um ataque, se for um ataque em área o dano é anulado para todos que seriam atingidos.

- A anulação deve ser decidia antes do dano ser rolado.
- A origem do dano deve estar em seu alcance. ( caso tenha reações você pode se mover para chegar até ela.)
